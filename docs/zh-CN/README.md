## 安装

1. 将Deep Go克隆到本地，下载zip或使用`GitHub Desktop`进行克隆
2. 将Deep Go文件夹添加到系统变量`PYTHONHOME`
3. 使用Anaconda安装依赖，推荐创建新的虚拟环境。

```cmd
conda install --yes --file requirements.txt
```

## 使用

Deep Go 的使用非常简单

```python
import deepgo as dg
```
